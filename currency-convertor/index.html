Currency Convertor Privacy Policy

Last updated: 2026-02-02

Overview

  • This extension detects prices on web pages and shows inline currency conversions.
  • It does not collect or sell personal information.

Data we access

  • Page text around prices to detect amounts and currency codes. This processing happens locally in your browser and is not sent off device.
  • The active tab URL is read only to show and toggle the per-site enable/disable setting in the popup.

Data we store (locally on your device)

  • Selected target currencies.
  • Per-site enabled/disabled settings.
  • Cached exchange rates (stored for up to 1 hour).

Network requests

  • Exchange rates and currency names are fetched from https://api.frankfurter.app.
  • Requests include only currency codes (base and target). We do not send page content, URLs, or user identifiers.

Data sharing

  • We do not share, sell, or rent user data.

Permissions use

  • storage: Save user settings and cache exchange rates locally.
  • activeTab: Read the current tab URL to display and toggle the "This site" setting.
  • host permission https://api.frankfurter.app/*: Fetch exchange rates and currency names.
  • content scripts on all sites: Detect and annotate prices on any website you visit.
